# Процесс сборки и запуска:
```bash
cmake -B build
cd build && make
./main
```
*main* - исполняемый файл, лежащий в директории *build* по окончанию сборки. Выводит в терминал сумму массива.

# Тип массива
По умолчанию используется тип *double*. Чтобы использовать *float* необходимо добавить флаг во время сборки:
```bash
cmake -B build -DFLOAT=ON
cd build && make
./main
```

# Выводы 
Сумма *double*-массива равна **-6.76916e-10**.

Сумма *float*-массива - **-0.0277862**.
