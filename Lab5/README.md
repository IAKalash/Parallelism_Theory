# Лабораторная работа №5

## Примеры запуска:

### Однопоточный режим
```bash
python3 lab.py --source input.mp4 --output out.mp4 --threads 1
```

### Многопоточный режим (8 потоков)
```bash
python3 lab.py --source input.mp4 --output out.mp4 --threads 8
```

### Обработка видеопотока с камеры
```bash
python3 lab.py --source 0 --display --threads 8
```
